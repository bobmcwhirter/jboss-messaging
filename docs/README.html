JBoss Messaging 1.2.0.GA Release Notes


March 01 2007


JBoss Messaging is a new JBoss JMS 1.1-compliant JMS provider, aimed to replace JBossMQ.  This is the first General Availability (GA) clustering release. 1.2.0.GA includes full clustering, with load balancing and transparent failover. An extensive overview of its features, as well as installation and configuration instructions are available in the "JBoss Messaging 1.2 User Guide". The guide is bundled with this release and is provided in HTML, HTML-single and PDF formats.  The guide is also available on-line at  http://labs.jboss.com/portal/jbossmessaging/docs/index.html

Summary of Changes

A complete list of changes introduced since the 1.0.1.GA release, is available below.
