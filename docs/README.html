JBoss Messaging 1.2.0.Beta1 Release Notes


December 20 2006


JBoss Messaging is a new JBoss JMS 1.1-compliant JMS provider, aimed to replace JBossMQ.  The curent release is the first Beta release. It includes the majority of the clustering features to be available in 1.2.0.GA. An brief overview of the clustering features and installation instructions are available in the "Clustering Guide". The guide is bundled with this release and is provided in HTML, HTML-single and PDF formats.  The guide is also available on-line at  http://labs.jboss.com/portal/jbossmessaging/docs/index.html

Warning
The current JBoss Messaging distribution is a preview beta release. We strongly advise against using it in a production environment.
