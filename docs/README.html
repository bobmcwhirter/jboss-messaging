Release Notes - JBoss Messaging - Version 2.0.0 Alpha


1st June 2008

These are the release notes for JBoss Messaging 2.0.0 Alpha

For full description of the contents please see the JBoss Messaging project JIRA.

This release is a preview release for forthcoming JBoss Messaging 2.0.0. It contains major changes from 1.4 including a Journal based persistence approach and a new transport that leverages Apache MINA.

Please note that this is an alpha release and we avoid against using it in production.

For more info on what this release has to offer refer to the user manual. Enjoy!! Release Notes - JBoss Messaging - Version 2.0.0 Alpha
